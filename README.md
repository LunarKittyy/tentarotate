# Tenta Rotator

Fixar upp-och-ner-vända tentasidor. Behåller första sidan, roterar resten 180°.

## Användning

Öppna `index.html` i en webbläsare. Släpp en PDF. Ladda ner den fixade versionen.

Allt körs lokalt — inget laddas upp.

## Varför

Inskannade tentor från lärare kommer ofta tillbaka med första sidan rätt men alla följande sidor upp och ner. Det här fixar det.
